# vuedemo-mpacc

## 用到的知识
 - vue.js2.0
 - vue-router
 - element-ui
 - webpack打包
 - 组件模块化
 
## 如何运行
 - git clone https://github.com/fshwc/vuedemo-mpacc.git
 - cd vuedemo-mpacc
 - cnpm install
 - npm run dev
 - 浏览器打开localhost:8080/mpacc
 
## 简要介绍
用vue相关知识重构了校内网，页面跳转用了vue-router
后台管理页面用了element-ui，美化页面
